# 🛒 E-Commerce Sales Dashboard

An interactive sales analytics dashboard built with Python, Streamlit, and Plotly. Designed to help business stakeholders explore 100,000+ e-commerce orders through dynamic filters and real-time visualizations — no spreadsheet required.

## 📊 Features

- **6 KPI Cards** — Total Revenue, Total Orders, Avg Order Value, Avg Rating, Return Rate, and Avg Delivery Days, all updating live based on filters
- **Revenue Over Time** — Area chart with switchable Monthly / Quarterly / Yearly grouping
- **Sales Breakdown** — Revenue or Quantity by Product Category (bar) and Region (donut chart)
- **Payment & Returns** — Revenue by payment method and return rate by product category
- **Product & Customer Insights** — Price vs Revenue scatter plot and customer rating distribution
- **Discount Impact** — Dual-axis chart showing order volume and average revenue across discount levels
- **Delivery Performance** — Average delivery days by region and full delivery distribution histogram
- **Raw Data Viewer** — Toggle-able table with full-text search and CSV export

### Sidebar Filters
Every chart responds in real time to:
- Date range picker
- Region (multiselect)
- Product Category (multiselect)
- Payment Method (multiselect)
- Order Type (All / Non-returned / Returned)
- Discount % range slider

## ⚙️ Getting Started (Run Locally)

### 1. Clone the repository
```bash
git clone https://github.com/your-username/sales-dashboard.git
cd sales-dashboard
```

### 2. Install dependencies
```bash
pip install -r requirements.txt
```

### 3. Update the data path
Open `Dashboard.py` and update the `DATA_PATH` variable to point to your local CSV file:
```python
DATA_PATH = "E-Commerce Sales Data.csv"
```
If the CSV is in the same folder as `Dashboard.py`, the filename alone is sufficient.

### 4. Run the app
```bash
streamlit run Dashboard.py
```
The dashboard will open automatically at `http://localhost:8501`

## 📈 Dataset

This dashboard uses a **synthetic e-commerce dataset** with 100,000 rows and the following columns:

| Column | Description |
|---|---|
| `order_id` | Unique order identifier |
| `customer_id` | Unique customer identifier |
| `product_category` | Category of the product ordered |
| `product_price` | Unit price of the product |
| `quantity` | Number of units ordered |
| `order_date` | Date the order was placed |
| `region` | Geographic region of the order |
| `payment_method` | Payment method used |
| `delivery_days` | Number of days to deliver |
| `is_returned` | Whether the order was returned (0/1) |
| `customer_rating` | Customer satisfaction rating (1–5) |
| `discount_percent` | Discount applied to the order |
| `revenue` | Total revenue generated by the order |

## 🚀 Deployment

This app is deployed on **Streamlit Cloud** (free tier).

To deploy your own copy:
1. Fork this repository
2. Go to [share.streamlit.io](https://share.streamlit.io)
3. Connect your GitHub account
4. Select this repo and set the main file to `Dashboard.py`
5. Click **Deploy**
